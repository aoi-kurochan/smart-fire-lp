# スマートFIRE LP

会社員を続けながら、AIを味方に「収入の柱」をもう一本 ——
「スマートFIRE」を伝えるメルマガ登録用のランディングページです。

## 構成

- `index.html` — LP本体（単一ファイル完結／画像はファイル内に埋め込み済み）

## ローカルでのプレビュー

このフォルダで簡易サーバーを起動し、ブラウザで開きます。

```bash
python3 -m http.server 8123
# → http://127.0.0.1:8123/index.html
```

`index.html` をブラウザで直接開いてもプレビューできます。
